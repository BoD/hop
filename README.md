# hop

A 'Spotlight' / 'Alfred' like launcher tool for MacOS, written in Kotlin/Compose Multiplatform.

Work in progress!

## Why?

- Because I can
- I've been an Alfred user for a long time, and I generally love it, but I've recently switched my main browser to Brave, and Alfred's
  bookmark search doesn't work with Brave (or other Chromium-based browsers). Tried Raycast, but it's not my cup of tea. Well how hard can
  it be, right? And the ability to write plugins in Kotlin is the best.
